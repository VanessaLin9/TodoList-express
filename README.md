# TodoList-express
----------------------------
由Express 及 Node.js 製作的備忘錄
使用MongoDB資料庫
***
## Installation-安裝說明

1. clone本專案
2. express install
3. 安裝 nodemon 
4. 安裝mongoDB
5. 安裝RoBo 3T
6. npm i mongoose
7. 執行 nodemon app.js
8. 開啟瀏覽器, 輸入網址 [http://localhost:3000](http://localhost:3000)
***
## Features-專案功能

+ 使用者可以在todo list 中檢視項目
+ 使用者可以在todo list 中增加項目
+ 使用者可以在todo list 中修改項目
+ 使用者可以在todo list 中刪除項目
